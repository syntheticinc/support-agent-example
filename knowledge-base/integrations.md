# Acme Integrations

Acme connects with the tools your team already uses. Supported native integrations:

- **Slack** — task notifications and slash commands.
- **GitHub** and **GitLab** — link commits and pull requests to tasks.
- **Jira** — two-way issue sync.
- **Google Drive** and **Figma** — attach files and designs.
- **Zapier** — connect to 5000+ apps.

More integrations are added regularly. If you need a custom integration, use the Acme REST API.
