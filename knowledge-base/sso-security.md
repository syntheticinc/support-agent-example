# SSO and Security

Single Sign-On (SSO) via SAML 2.0 is available **only on the Business plan**. Supported identity providers include Okta, Azure AD, and Google Workspace.

Two-factor authentication (2FA) is available on all plans, including Free. All data is encrypted at rest using AES-256 and in transit using TLS 1.3.

SCIM user provisioning is a Business-plan feature.
